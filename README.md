# NeuroCanvas
NeuroCanvas is an EEG-driven generative art system that transforms brainwave activity into real-time visual expression. Using Muse EEG signals, the system estimates emotional state as valence and arousal, then uses those values to control dynamic visuals in TouchDesigner.
